# Simple Aegon Calculator

## Building and running
./mvnw spring-boot:run

## Testing
./mvnw test

## REST Endpoints (with a/b parameter)
Available on http://localhost:8080 after starting.

 - /add
 - /subtract
 - /divide
 - /multiply
